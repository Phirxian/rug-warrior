int bumper_detection ( )

Compensate the fact that there's a difference between instantaneous detection and the actual test The main change is to increase the accuracy of the detection of the front bumper: We insert an active temporisation when bumper() returns a front detection (left && right); See if the front is touched (will probably need to increase the angle of the hit), to contrary bumper() CAUTION: LAP OF THE DETECTION IS NOT LONG -> OVERRIDE LEFT & RIGHT, LOW -> use bumper()

Returns
the touched bumper else 0 if no bumper(s) is/are touched
